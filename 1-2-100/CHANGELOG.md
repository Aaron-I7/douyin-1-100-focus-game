# 更新日志

## v2.0.1 - UI显示修复 (2024-03-15)

### 🐛 Bug修复

#### Canvas DPI缩放问题
- **问题**: UI元素完全不可见，只显示黄色背景
- **原因**: `ctx.scale(dpr, dpr)` 在初始化时就被应用，导致UIManager绘制的所有元素被缩放到屏幕外
  - Canvas逻辑尺寸: 393×852
  - Canvas物理尺寸: 1179×2556 (DPI=3)
  - UIManager在逻辑坐标绘制，但context已被缩放3倍
- **解决方案**:
  - 移除初始化时的DPI缩放
  - 仅在开始游戏时应用DPI缩放（用于Voronoi高清渲染）
  - 在渲染UI前重置canvas变换矩阵 `ctx.setTransform(1, 0, 0, 1, 0, 0)`
- **影响**: 所有UI界面（开始、模式选择、难度选择、结果）现在都能正常显示
