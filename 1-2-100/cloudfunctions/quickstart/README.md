# quickstart cloud functions

This folder contains daily-challenge APIs for the game:
- `get_open_id`
- `get_daily_status`
- `start_game_session`
- `submit_level2_result`
- `request_revive`
- `get_province_leaderboard`
- `get_home_stats`

Core rules:
- no progress save/load
- only level-2 is submitted
- revive max 2 per session, +180s each
