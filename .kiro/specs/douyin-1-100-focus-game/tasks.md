# 实现计划：抖音 1-100 专注力训练游戏（重构版）

## 概述

本实现计划基于现有代码进行重大重构，移除黑暗模式和连击系统，实现渐进式关卡系统（第一关 1-10，第二关 1-100，通关后解锁自选模式），集成抖音云用户系统和平台能力，并采用独特的视觉设计。

## 重大变更

1. **移除黑暗模式**: 删除手电筒光照效果、连击系统、黑暗模式相关代码
2. **实现关卡系统**: 第一关 1-10 → 第二关 1-100 → 自选模式（需通关解锁）（每天都需要走这个流程）
3. **集成抖音云服务**: 用户登录、云端数据存储、进度同步
4. **集成抖音平台能力**: 分享、激励视频广告、数据分析
5. **重构视觉设计**: 使用 frontend-design skill 创建独特配色和动画效果
6. **更新游戏流程**: 登录 → 加载进度 → 主菜单 → 关卡/自选模式

实现策略：先移除旧功能，再实现新系统，最后集成平台能力和视觉设计。
