

我想做一个抖音小游戏，名字叫“1-100”，是一个舒尔特方格训练的小程序；
主要界面：
	1、登录过程界面（不需要用户名、密码），只是一个登录过程的界面；
	2、首页
	- 符合游戏特点的背景图片；
	- 开始游戏按钮（按钮上需要显示“今日剩余免费次数”）；
	- 右上方有一块黑板显示进入挑战人生、今日通过人数；
	- 主版面显示通关人数所在省份的排名，不要使用纯文字，要实现部分通过人的头像；
	3、第一关游戏界面
	- 顶部片下方显示剩余时间，但是第一关不显示（不要被手机眉遮挡）；
	- 游戏界面，9个数字随机分割，最好是不规则的分割，立体方块，点击有下沉感，点错有反馈；
	- 支持退出
	4、第一关完成后，页面飘出“最后一关”的字样，同时在同一个界面渲染1~100的数字方格；
	5、第二关游戏界面。同第一关，只是从1~9变成1~100，同时添加10分钟倒计时；
	6、通关界面，弹窗提示恭喜通关，返会首页，同时首页通过人板块，添加个人头像显示；
